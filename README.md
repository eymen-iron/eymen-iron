### About

Full-stack developer with a strong backend focus, working across the entire stack — from raw SQL and Nginx configs to SSR frontends and container orchestration. I care about clean architecture, performance, and security at every layer.

Day-to-day, I build with **Go** and **TypeScript**. I have production experience with **PHP/Laravel** and **Python/Django**, and I'm currently exploring **Rust** for systems-level work and **WebGL/GLSL** for graphics programming.

When I'm not building, I'm breaking — actively practicing **offensive security** with tools like Burp Suite, SQLMap, and FFUF.
